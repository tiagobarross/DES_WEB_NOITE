# 🏋️ FitConsult

FitConsult é uma **Single Page Application (SPA)** desenvolvida para a disciplina de **Desenvolvimento Web**, com o objetivo de apresentar e comercializar planos de consultoria fitness. A aplicação oferece uma interface moderna e intuitiva para que os usuários possam conhecer os serviços, selecionar um plano e entrar em contato através de um formulário.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## ✨ Funcionalidades

* Apresentação dos serviços de consultoria fitness
* Exibição de diferentes planos disponíveis
* Seleção de planos
* Formulário de contato para potenciais clientes
* Design responsivo para diferentes dispositivos
* Navegação em página única (Single Page Application)

## 📸 Objetivo do Projeto

O projeto foi desenvolvido com fins acadêmicos para aplicar conceitos fundamentais de desenvolvimento web, incluindo estruturação semântica com HTML5, estilização com CSS3 e interatividade utilizando JavaScript.
